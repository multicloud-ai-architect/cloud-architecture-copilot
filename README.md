## Overview

Cloud Architecture Copilot — open source AI assistant for architects, VS Code + JetBrains extensions, MCP servers. This project is part of the MAIA — Multi-Cloud AI Architect portfolio, providing reference architectures and reusable building blocks for cloud architects working across AWS, Azure, and GCP.
